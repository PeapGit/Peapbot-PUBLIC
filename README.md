# Peapbot-PUBLIC

# English (US)

This bot is my personal bot for doing stuff with friends and such. The documentation will be translated in three separate languages. Each in separate folders labeled English, German, and Hindu. 

To get started put your token.txt in the same folder as either the main.py (for guilds) or main2.py for apps. 

# German (Deutsch)

Dieser Bot ist mein persönlicher Bot, um Dinge mit Freunden und Ähnliches zu erledigen.
Die Dokumentation wird in drei separate Sprachen übersetzt.
Jede Sprache befindet sich in einem eigenen Ordner mit den Bezeichnungen English, German und Hindi.

Um zu beginnen, lege deine token.txt in denselben Ordner wie entweder main.py (für Guilds) oder main2.py (für Apps).
